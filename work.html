Weather App

HTML · CSS · JavaScript · API

I developed this weather application using HTML, CSS, JavaScript, and an API. The application's user interface is built using HTML and CSS, providing a clean and intuitive design. I focused on creating a visually appealing layout that is easy to navigate, allowing users to quickly find the information they need.

To fetch the weather data, I integrated an API into the application using JavaScript which enables users to retrieve real-time weather updates for various locations.

This project was fabulous learning as I overcame some roadblocks with the API, explore my design abilities and test my JavaScript knowledge.
