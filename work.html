World Clock App

HTML · CSS · JavaScript · API · Responsive

I developed this modern and responsive world clock application that showcases the current time in multiple cities around the world. It is built using HTML, CSS, JavaScript and moment.js API, which handles accurate time calculations and timezone conversions in real-time.


Users can easily navigate through a list of cities and view the current time for each location. In addition, they have the option to choose their current location, which enhances the personalized experience of the application.


This was an exciting and rewarding project that challenged me to integrate multiple technologies, allowing me to create a functional application while gaining knowledge of new coding concepts.
